## DUBBO代码案例

### 项目概述

**api 模块**

提供接口供消费者使用和生产者实现

provider-01与consumer-01

**springboot整合dubbo**

- 直连模式
- zookeeper注册中心模式

**provider-02与consumer-02**

springcloud整合dubbo

- zookeeper注册
- nacos注册
- 容错、服务降级、负载均衡等测试
